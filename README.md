# 🔴 Programação Orientada a Objetos (POO)

> Construindo a base da orientação a objetos com Java.

---

> [!ABSTRACT]
> Repositório acadêmico da disciplina **Programação Orientada a Objetos (POO)** da FATEC.
>
> Durante a disciplina foram desenvolvidos exercícios e projetos utilizando Java, abordando os principais conceitos da orientação a objetos, tratamento de exceções, coleções e modelagem de sistemas.
>
> O objetivo foi compreender como estruturar aplicações utilizando boas práticas de programação orientada a objetos.
