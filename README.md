# 🔴 Programação Orientada a Objetos (POO)

Construindo a base da orientação a objetos com Java.

## 📖 Sobre

Repositório acadêmico da disciplina de Programação Orientada a Objetos (POO) da FATEC.

Durante a disciplina foram desenvolvidos exercícios e projetos utilizando Java, abordando conceitos como:

- Classes e Objetos
- Encapsulamento
- Herança
- Polimorfismo
- Exceções
- Collections
- Modelagem de Sistemas

## ☕ Tecnologias

| Tecnologia | Aplicação |
|------------|-----------|
| Java | Linguagem principal |
| IntelliJ IDEA | Desenvolvimento |
| UML | Modelagem |
| Collections Framework | Estruturas de dados |
